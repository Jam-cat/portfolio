当院について

当院は痛みの少ない治療、身体に負担のかからない治療をするようにしておりますのでお子様も安心して通院して頂けます。
また、健康な歯を保つには何より予防が重要だと考えております。歯磨き指導など予防歯科のためのアドバイスも積極的にさせて頂いております。

託児サービス

当院では、保育士資格を持つスタッフが常駐しておりますので、お子様連れの保護者の方も定期的な通院がしやすくなっております。
また、保護者の方とお子様の治療が同時に済むので助かるというお声もたくさん頂いております。

診療案内

一般診療

「歯が痛い」「食べるときに歯がしみる」「歯ぐきが腫れている」そう感じていても、治療せずそのままにしている方も多いかもしれません。しかし、むし歯は悪化すればするほど治すのに時間がかかり痛みや治療費用も増大してしまいます。おかしいな？と思ったら早めの受診をおすすめします。

口腔外科

口腔外科とは、文字通りお口の中の外科治療を指します。親知らずの抜歯や、顎関節症の治療、スポーツによる外傷の治療を行います。不安に感じたなら、すぐに治療を始めましょう。

小児歯科

治療中のお子さんの不安を取り除くために、当院では液晶モニターにてアニメを観ることができます。 定期的に「歯のおはなし会」を開催し、お子様がはみがきを楽しいと思えるようなきっかけづくりにも力をいれております。
